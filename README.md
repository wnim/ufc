# 🤜💥😵 UFC Finishes

Spoiler-free UFC finish tracker. See which fights ended in a finish — without learning who won or how.

**[ufcfinishes.netlify.app](https://ufcfinishes.netlify.app)**

## What it does

- Shows recent UFC main-card fights (last 3–24 months, your choice)
- Filters to finishes only by default (toggle decisions back on if you want)
- Randomizes which fighter appears left/right so you can't infer the winner
- Counts update in the background so you know what's waiting before you click

Data scraped from Wikipedia.
